Why S3 Backpack?

🎨

Beautiful TUI

Colorful, interactive terminal UI with spinners and real-time status updates

⚡

Zero Setup

Auto-installs MinIO client if missing. Just run and go

✅

Verify & Validate

Verifies connection and credentials before any download starts

📦

Multi-Select Buckets

Choose one or many buckets. Each gets downloaded and compressed

🔒

Secure

Supports self-signed TLS, API signatures v4 and v2, custom regions

🐧

Cross-Platform

Works on macOS and Linux with Node.js 18+

Quick Start

1

Run the command

npx github:AlmossaidLLC/s3-backpack-cli
2

Enter your S3 credentials

Endpoint URL, access key, and secret key

3

Select buckets

Use Space to select, Enter to backup

4

Done!

Buckets downloaded and zipped in your chosen directory
